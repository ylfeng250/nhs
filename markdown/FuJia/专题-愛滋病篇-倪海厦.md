何大一警告：中国正面临三大病毒威胁
【中央社 记者林于国香港二十四日电】
中央研究院院士何大一指出，中国正面临SARS、艾滋病及流感病毒三大威胁。其中，威
胁最大的流感病毒将再一次在中国大规模流行。他并建议北京当局必须对可能发生的生物
恐怖威胁，建立有效的全国公共卫生监察机构。
何大一昨天应邀在广州专题演讲时，赞扬香港处理禽流感及SARS工作及时，医疗体系较
完善，并呼吁香港市民对于近日来三十多名明爱医院儿童发烧事件无需过度紧张。

他分析中国面对突发的公共卫生事件的优势是具有处理SARS经验，及政府具有调动、统
一防御的能力。但他提醒，流感若在亚洲南部爆发，中国将受很大影响，而且中国缺乏国
家性准备计划，公共监督体系不完全。
此外，贫困地区医疗不足，无法立即生产足够应付流感疫苗的能力，也是中国需解决的问
题。
何大一并指出，中国经济活动日益活跃，国际事务增多，必须具有应对可能发生的生物恐
怖威胁的能力。
他建议，长期目标要增加基础研究的基金投入和改进卫生系统的组织建设。短期内先要对
重要的专业人员制订培训计划。此外，也应升级疾病防疫系统，增加透明度。

[医学动态]艾滋病新疗法亦使卡波济氏肉瘤病例急剧下降--欧洲研究
| [路透亚特兰大电]  |     | 根据周一发表的一份欧洲研究，由于抗逆转录病毒药物的使用，患卡波 |     |     |     |     |     |     |     |     |     |
| ---------- | --- | ------------------------------- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
济氏肉瘤(Kaposi sarcoma)的艾滋病毒(HIV)感染者数量已急剧下降。
卡波济氏肉瘤曾是一种罕见癌症但在艾滋病(AIDS)流行初期，它成了艾滋病人的标记性疾
病。
根据对近一万名HIV患者的研究显示，1994-2003年间，卡波济氏肉瘤的年病例数减少了
39%。本项研究是由英国皇家自由医院  (Royal  Free  Hospital)、伦敦大学学院(University
College in London)和另外几家欧洲医院及卫生机构主办的。
卡波济氏肉瘤最早以褐色皮肤病变的形式出现，不过也可以发生在肺部、肝脏和其它内脏
器官。在艾滋病于1981年浮现之前，此种癌症主要发生在地中海地区的年长男子。
但到了1980年代，它成了折磨艾滋病人的最常见疾病之一。
又过了10年，一种称为高效抗逆转录病毒治疗(HAART)的多药物合并疗法诞生，让医生
拥有了一个治疗卡波济氏肉瘤的强大新武器，从而让这种导致许多艾滋病人死亡的病症病
例数降低。
此前曾有非正式数据和小型研究显示，随着新药物压制了病人血液的HIV水平，并让病人
的免疫系统恢复，卡波济氏肉瘤病例数正在下降。
不过本次欧洲大型研究首次确切表明，上述疗法和卡波济氏肉瘤病例数的下降有关。研究
结果发表在美国癌症学会期刊《癌症》(Cancer)的5月10日网络版上。(完)

手术后廿个月 病毒都不见了
「柏林慈善医院」医师惠特表示，本案例的病患是现年四十二岁的美国男子，现居柏
林市，同时患有艾滋病及白血病。要治疗白血病，最好的方法便是骨髓移植；医院小组特

移植到病患体内。

疗团队都有病原复发的心理准备，但不论院方或者医学院专家都表示，检验结果显示，患
者的骨髓、血液及其它器官组织，ＨＩＶ病毒都告「清除完毕」。

年间便曾实验过卅二次。一九九九年，《医学假说》期刊有专文审核那些实验的成果，文中

ＣＣＲ５受体，免于遭到ＨＩＶ病毒攻击。
