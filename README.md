# 倪海厦中医文集（nhs）

整理倪海厦先生（Ni Hai-Sha, 1954—2012）的中医讲稿、医案与论述，全部以 Markdown 形式归档，便于阅读、检索与全文搜索。

> 倪海厦先生系美国汉唐中医学院创办人，毕生致力于传统中医的传承与发扬，以「天纪、人纪、地纪」三纪之学行世。本仓库收录其授课讲稿、临床医案与专题论述等一手资料。

---

## 仓库结构

```
markdown/
├── TianJi/   # 天纪：紫微斗数、人间道、地脉道、天机道
├── RenJi/    # 人纪：针灸、神农本草经、黄帝内经、伤寒论、金匮要略
├── DiJi/     # 地纪：地纪日记
├── YiAn/     # 医案：临床医案、诊疗日志、文集
└── FuJia/    # 附加：专题论述、方剂讲解、经方论治等
```

## 使用说明

- 所有文档均为 UTF-8 编码的 Markdown，可直接在 GitHub、VS Code、Obsidian 等任意 Markdown 阅读器中查看。
- 文件命名遵循 `{分类}-{主题}-{作者/版本}.md` 的约定，便于区分视频稿、彩版、简版与图文笔记等不同版本。
- 全文检索建议使用 `grep`、`rg` 或阅读器自带的搜索功能。
